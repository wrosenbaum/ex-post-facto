Thinking classes

What do we need?

  1. class Node representing enclosure
    • stores (reference to) an Animal
    • stores (reference to) next Node
  2. class Zoo
    • stores (reference to) first Node (if any)
    • stores (reference to) last Nodes (if any)
    • implements desired operations:
      • feed Animals
      • add an Animal
      • remove an Animal
      • …
